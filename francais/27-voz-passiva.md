# Lição 27 — A Voz Passiva e as Construções Impessoais

---

## Objetivos

Ao final desta lição, você será capaz de:

- Transformar frases da voz ativa para a voz passiva em francês
- Conjugar a voz passiva em todos os tempos verbais já estudados
- Fazer a concordância correta do particípio passado na voz passiva
- Usar **on** como alternativa mais natural à voz passiva
- Distinguir quando usar **par** e quando usar **de** na voz passiva
- Empregar construções impessoais com **il est** + adjetivo + modo correto (indicativo ou subjuntivo)

## Explicação

### Parte 1: Voz Ativa vs. Voz Passiva — O Conceito

Raphael, a voz passiva funciona de maneira muito parecida em francês e em português. A diferença principal é **quem pratica** e **quem sofre** a ação.

#### Na voz ativa:
O sujeito **faz** a ação.

> Le développeur écrit le code. (O desenvolvedor escreve o código.)

#### Na voz passiva:
O sujeito **sofre** a ação. Quem faz a ação vira o **agente** (introduzido por **par**).

> Le code est écrit par le développeur. (O código é escrito pelo desenvolvedor.)

#### A fórmula da voz passiva

```
SUJEITO (que sofre a ação) + ÊTRE (conjugado no tempo desejado) + PARTICIPE PASSÉ + par + AGENTE (quem faz a ação)
```

Pense assim: é como o passé composé com **être**, mas o verbo **être** pode ser conjugado em **qualquer tempo**.

**Mais exemplos com o seu cotidiano:**

| Voz Ativa | Voz Passiva |
|-----------|-------------|
| Katya corrige les exercices. | Les exercices **sont corrigés** par Katya. |
| L'équipe développe la fonctionnalité. | La fonctionnalité **est développée** par l'équipe. |
| Le professeur de jiu-jitsu enseigne la technique. | La technique **est enseignée** par le professeur de jiu-jitsu. |

> **Dica:** Para transformar uma frase ativa em passiva, siga três passos: (1) o COD vira sujeito, (2) o verbo vira être + participe passé, (3) o sujeito original vira agente com **par**.

### Parte 2: A Voz Passiva em Todos os Tempos

Raphael, a boa notícia é que a estrutura é sempre a mesma: **être (conjugado no tempo) + participe passé**. Você já conhece todos esses tempos! Vamos usar um exemplo do seu trabalho:

#### Tabela completa de tempos

| Tempo verbal | Être conjugado | Exemplo completo |
|-------------|----------------|------------------|
| **Présent** | est | Le bug **est corrigé** par le développeur. |
| **Passé composé** | a été | Le bug **a été corrigé** par le développeur. |
| **Imparfait** | était | Le bug **était corrigé** par le développeur chaque semaine. |
| **Plus-que-parfait** | avait été | Le bug **avait été corrigé** avant la mise à jour. |
| **Futur simple** | sera | Le bug **sera corrigé** demain. |
| **Futur proche** | va être | Le bug **va être corrigé** bientôt. |
| **Conditionnel présent** | serait | Le bug **serait corrigé** si on avait le temps. |
| **Conditionnel passé** | aurait été | Le bug **aurait été corrigé** si on avait eu le temps. |

#### Exemplos detalhados por tempo

**Présent** — ação habitual ou estado atual:
- La base de données **est mise à jour** chaque nuit. (O banco de dados é atualizado toda noite.)
- Les tests **sont exécutés** automatiquement. (Os testes são executados automaticamente.)

**Passé composé** — ação concluída no passado:
- Le code **a été déployé** ce matin. (O código foi implantado esta manhã.)
- La fonctionnalité **a été validée** par l'équipe QA. (A funcionalidade foi validada pela equipe de QA.)

**Imparfait** — ação habitual ou contínua no passado:
- Avant, les bugs **étaient corrigés** manuellement. (Antes, os bugs eram corrigidos manualmente.)
- Les réunions **étaient organisées** tous les lundis. (As reuniões eram organizadas todas as segundas.)

**Plus-que-parfait** — ação anterior a outra no passado:
- Quand je suis arrivé, le problème **avait déjà été résolu**. (Quando eu cheguei, o problema já tinha sido resolvido.)
- La mise à jour **avait été annulée** avant le déploiement. (A atualização tinha sido cancelada antes da implantação.)

**Futur simple** — ação futura:
- Le nouveau module **sera développé** le mois prochain. (O novo módulo será desenvolvido mês que vem.)
- Les utilisateurs **seront informés** par e-mail. (Os usuários serão informados por e-mail.)

**Conditionnel présent** — hipótese ou possibilidade:
- Ce bug **serait corrigé** facilement avec un patch. (Esse bug seria corrigido facilmente com um patch.)
- Le projet **serait terminé** plus vite avec un autre développeur. (O projeto seria terminado mais rápido com outro desenvolvedor.)

**Conditionnel passé** — hipótese no passado:
- Le bug **aurait été évité** si on avait fait des tests. (O bug teria sido evitado se tivéssemos feito testes.)

> **Atenção:** No passé composé passivo, **être** fica no passé composé (a été), então você tem **dois participes**: a **été** + corrig**é**. Não se assuste com essa construção dupla!

### Parte 3: A Concordância na Voz Passiva

Na voz passiva, o particípio passado **sempre concorda com o sujeito**, porque o auxiliar é **être** (lembra da regra do passé composé com être?).

#### Regras de concordância

| Sujeito | Concordância | Exemplo |
|---------|-------------|---------|
| Masculino singular | — | Le bug est corrig**é**. |
| Feminino singular | +e | La fonctionnalité est développ**ée**. |
| Masculino plural | +s | Les bugs sont corrig**és**. |
| Feminino plural | +es | Les données sont export**ées**. |

**Exemplos do cotidiano:**

- La pull request **a été approuvée** par le tech lead. (A pull request foi aprovada pelo tech lead.)
- Les tâches **ont été assignées** à l'équipe. (As tarefas foram atribuídas à equipe.)
- Le match de jiu-jitsu **a été gagné** par mon adversaire. (A luta de jiu-jitsu foi ganha pelo meu adversário.)
- Les nouvelles fonctionnalités **sont testées** avant le déploiement. (As novas funcionalidades são testadas antes da implantação.)

> **Dica:** Se você errar a concordância na voz passiva, um francês vai perceber. Sempre se pergunte: o sujeito é masculino ou feminino? Singular ou plural? E ajuste o particípio.

### Parte 4: A Alternativa com "On"

Raphael, aqui está uma informação essencial: **na fala cotidiana, os franceses raramente usam a voz passiva**. Eles preferem usar **on** como sujeito indefinido. É muito parecido com o nosso "a gente" ou com frases sem sujeito definido.

#### Comparação: passiva vs. "on"

| Voz passiva (formal/escrita) | Com "on" (oral/cotidiano) |
|-----------------------------|--------------------------|
| Le français est parlé ici. | **On parle** français ici. |
| Le bug a été réparé. | **On a réparé** le bug. |
| La porte a été fermée. | **On a fermé** la porte. |
| Les résultats seront annoncés demain. | **On annoncera** les résultats demain. |
| Il m'a été dit que... (muito formal) | **On m'a dit** que... |
| Le match a été annulé. | **On a annulé** le match. |

**Exemplos personalizados:**

- **On** utilise C# pour développer Jure. (= C# est utilisé pour développer Jure.)
- **On** m'a demandé de corriger ce bug. (= Il m'a été demandé de corriger ce bug.)
- **On** joue au jiu-jitsu trois fois par semaine. (= Le jiu-jitsu est pratiqué trois fois par semaine.)
- **On** dit que Solo Leveling est un bon anime. (= Il est dit que Solo Leveling est un bon anime.)

> **Dica:** Na conversa com a Katya ou com amigos, use **on**. Na escrita formal (relatórios, e-mails profissionais, artigos), use a voz passiva. É a mesma diferença que temos em português entre "A reunião foi cancelada" (formal) e "Cancelaram a reunião" (informal).

### Parte 5: Quando NÃO Usar a Voz Passiva

Nem todo verbo pode ir para a voz passiva! A regra é simples:

> **Atenção:** Apenas verbos que têm um **COD (complemento de objeto direto)** podem ser transformados em voz passiva. Verbos intransitivos (sem COD) **não podem**.

#### Verbos que NÃO aceitam voz passiva

| Verbo | Por quê? |
|-------|----------|
| aller (ir) | intransitivo — não tem COD |
| venir (vir) | intransitivo |
| dormir (dormir) | intransitivo |
| mourir (morrer) | intransitivo |
| arriver (chegar) | intransitivo |
| partir (partir/sair) | intransitivo |
| tomber (cair) | intransitivo |
| rester (ficar) | intransitivo |

**Teste prático:** Pergunte-se: "O verbo responde à pergunta **quoi?** (o quê?) ou **qui?** (quem?)". Se sim, tem COD e pode virar passiva.

- Je mange **une pizza**. → **Quoi?** Une pizza. → COD! → La pizza est mangée par moi. (OK)
- Je dors. → **Quoi?** ... nada. → Sem COD. → Impossível na voz passiva. (ERRO)

#### Cuidado com os verbos que usam COI (objeto indireto)

Verbos que pedem **à** (COI) também **não podem** virar passivos:

- Je téléphone **à** mon collègue. → NÃO se pode dizer: ~~Mon collègue est téléphoné.~~
- J'obéis **à** mon professeur. → NÃO: ~~Mon professeur est obéi.~~

> **Dica:** Diferente do inglês, onde "He was told" e "She was given" são normais, em francês o COI nunca vira sujeito da passiva. "On lui a dit" é a forma correta.

### Parte 6: "Par" vs. "De" na Voz Passiva

Na maioria dos casos, usamos **par** para introduzir o agente. Mas em alguns casos específicos, usamos **de**.

#### Quando usar PAR

**Par** é usado quando o agente **faz uma ação concreta/dinâmica**:

- Le code est écrit **par** le développeur. (ação de escrever)
- Le match a été gagné **par** l'équipe brésilienne. (ação de ganhar)
- Le bug a été corrigé **par** Raphael. (ação de corrigir)
- La fonctionnalité sera testée **par** l'équipe QA. (ação de testar)

#### Quando usar DE

**De** é usado com verbos que expressam **estado, emoção, percepção** ou quando o resultado é um **estado estático**:

| Exemplo com DE | Tradução |
|---------------|----------|
| Il est **aimé de** tous. | Ele é amado por todos. |
| Elle est **respectée de** ses collègues. | Ela é respeitada pelos colegas. |
| La ville est **entourée de** montagnes. | A cidade é cercada por montanhas. |
| Le mur est **couvert de** graffitis. | O muro é coberto de grafites. |
| Il est **connu de** tout le monde. | Ele é conhecido por todo mundo. |
| La salle est **remplie d'**étudiants. | A sala está cheia de estudantes. |

> **Dica:** Pense assim: **par** = alguém **faz** algo ativamente. **De** = algo **está** em um estado. "O bolo foi feito **por** minha mãe" (par — ação) vs. "A mesa estava coberta **de** livros" (de — estado).

**Alguns verbos que frequentemente usam DE:**

aimer, adorer, respecter, détester, entourer, couvrir, remplir, accompagner, précéder, suivre (em sentido figurado), connaître

### Parte 7: Construções Impessoais

As construções impessoais começam com **il** (sujeito formal, sem significado real). Raphael, você já viu algumas delas nas lições de subjuntivo! Aqui vai um resumo completo:

#### Construções com INDICATIVO (certeza, constatação)

Estas expressam fatos, certezas ou informações relatadas. Usam o **indicatif**:

| Construção | Significado | Exemplo |
|-----------|-------------|---------|
| il est certain que | é certo que | Il est certain que **C# est** un bon langage. |
| il est sûr que | é seguro/certo que | Il est sûr que **le bug vient** du back-end. |
| il est évident que | é evidente que | Il est évident que **tu fais** des progrès en français. |
| il est clair que | é claro que | Il est clair que **cette API fonctionne** bien. |
| il paraît que | parece que (dizem que) | Il paraît que **Frieren est** un excellent anime. |
| il me semble que | me parece que | Il me semble que **le sprint se passe** bien. |

> **Dica:** **Il paraît que** expressa um boato, algo que ouvimos dizer. É como nosso "dizem que..." ou "parece que...". É sempre seguido de **indicativo**.

#### Construções com SUBJUNTIVO (dúvida, necessidade, possibilidade, julgamento)

Estas expressam opinião, necessidade, dúvida ou possibilidade. Usam o **subjonctif** (lições 21-22):

| Construção | Significado | Exemplo |
|-----------|-------------|---------|
| il est important que | é importante que | Il est important que tu **finisses** tes exercices. |
| il est nécessaire que | é necessário que | Il est nécessaire que l'équipe **fasse** des tests. |
| il faut que | é preciso que | Il faut que je **corrige** ce bug avant le sprint. |
| il est possible que | é possível que | Il est possible qu'il **pleuve** demain. |
| il semble que | parece que (incerteza) | Il semble que le bug **soit** dans le back-end. |
| il est dommage que | é uma pena que | Il est dommage que tu ne **puisses** pas venir. |
| il vaut mieux que | é melhor que | Il vaut mieux que nous **partions** tôt. |
| il est rare que | é raro que | Il est rare que le serveur **tombe** en panne. |

> **Atenção:** Note a diferença entre **il me semble que** (+ indicatif — mais certeza pessoal) e **il semble que** (+ subjonctif — mais dúvida). É um detalhe sutil, mas importante!

#### Construções com INFINITIVO (geral, sem sujeito específico)

Quando a frase é **geral** (sem sujeito específico), usamos **de + infinitif**:

| Construção | Exemplo |
|-----------|---------|
| il est important de | Il est important **de bien dormir**. |
| il est nécessaire de | Il est nécessaire **de tester le code**. |
| il est possible de | Il est possible **de travailler à distance**. |
| il est interdit de | Il est interdit **de fumer ici**. |
| il est difficile de | Il est difficile **de maîtriser le subjonctif**. |
| il est facile de | Il est facile **d'apprendre avec de bons exemples**. |

> **Dica:** Use **de + infinitif** quando a ideia é geral ("É importante dormir bem" — para todo mundo). Use **que + subjonctif** quando há um sujeito específico ("É importante que **tu** durmas bem" — para você).

**Exemplos personalizados:**

- Il est important **de** faire du sport. → Il est important **que tu fasses** du jiu-jitsu régulièrement.
- Il est nécessaire **de** déboguer le code. → Il est nécessaire **que Raphael débogue** le code avant le déploiement.
- Il paraît **que** Solo Leveling va avoir une saison 3. (Parece que Solo Leveling vai ter uma temporada 3.)
- Il est possible **que** la professeure Katya **soit** en vacances la semaine prochaine.
- Il est certain **que** le Hitchhiker's Guide **est** un livre génial.
