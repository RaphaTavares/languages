### Legenda

- **Particípio irregular** = não segue as regras -é / -i / -u
- **Radical FS irregular** = não usa o infinitivo como base
- *(être)* = usa être como auxiliar no PC
- *(regular)* = o radical do FS segue a regra normal (infinitivo ou infinitivo sem -e)

## 2. Os Padrões dos Particípios Irregulares

| Terminação | Verbos | Particípios |
|-----------|--------|------------|
| **-it** | faire, dire, écrire | fait, dit, écrit |
| **-is** | prendre, mettre | pris, mis |
| **-u** | voir, pouvoir, savoir, devoir, venir, tenir, boire, lire, courir, connaître, recevoir, pleuvoir, falloir | vu, pu, su, dû, venu, tenu, bu, lu, couru, connu, reçu, plu, fallu |
| **-ert** | ouvrir, offrir, souffrir, découvrir | ouvert, offert, souffert, découvert |
| **-é (irregular)** | être, naître | été, né |
| **-ort** | mourir | mort |

## 3. Os Padrões dos Radicais Irregulares do Futur Simple

| Padrão | Verbos | Radical |
|--------|--------|---------|
| **Radicais totalmente diferentes** | être → ser-, avoir → aur-, faire → fer-, aller → ir-, falloir → faudr- | |
| **Radical com -rr-** (consoante dobrada) | pouvoir → pourr-, voir → verr-, courir → courr-, mourir → mourr-, envoyer → enverr- | |
| **Radical com -dr-** | vouloir → voudr-, venir → viendr-, tenir → tiendr-, falloir → faudr- | |
| **Radical com -vr-** | devoir → devr-, savoir → saur-, recevoir → recevr-, pleuvoir → pleuvr- | |

## 4. DR MRS VANDERTRAMP — Verbos que usam ÊTRE no Passé Composé

Esses verbos usam **être** (e não avoir) como auxiliar no passé composé. O particípio concorda em gênero e número com o sujeito.

### O Acrônimo

| Letra | Verbo | Particípio | Exemplo |
|-------|-------|-----------|---------|
| **D** | Devenir (tornar-se) | devenu | elle est devenu**e** médecin |
| **R** | Revenir (voltar) | revenu | ils sont revenu**s** |
| **M** | Mourir (morrer) | mort | il est mort |
| **R** | Retourner (retornar) | retourné | nous sommes retourné**s** |
| **S** | Sortir (sair) | sorti | elle est sorti**e** |
| **V** | Venir (vir) | venu | je suis venu |
| **A** | Aller (ir) | allé | tu es allé**e** |
| **N** | Naître (nascer) | né | elle est né**e** en 1990 |
| **D** | Descendre (descer) | descendu | ils sont descendu**s** |
| **E** | Entrer (entrar) | entré | je suis entré |
| **R** | Rentrer (voltar pra casa) | rentré | il est rentré tard |
| **T** | Tomber (cair) | tombé | elle est tombé**e** |
| **R** | Rester (ficar) | resté | nous sommes resté**s** |
| **A** | Arriver (chegar) | arrivé | elles sont arrivé**es** |
| **M** | Monter (subir) | monté | je suis monté |
| **P** | Partir (partir) | parti | il est parti |
| + | Passer (passar) | passé | elle est passé**e** par ici |

### Visualização em Pares de Opostos

A maioria dos verbos DR MRS VANDERTRAMP vem em pares de movimento oposto:

```
        NAÎTRE ←――――→ MOURIR
        ARRIVER ←――――→ PARTIR
        ENTRER ←――――→ SORTIR
        MONTER ←――――→ DESCENDRE
        ALLER ←――――→ VENIR (REVENIR, DEVENIR)
        TOMBER ←――――→ RESTER
        RETOURNER ←――→ RENTRER
        PASSER (passagem de um lado a outro)
```

### As 3 Regras

1. **Sempre être:** aller, venir, naître, mourir, rester, tomber, devenir
2. **Être OU avoir** (muda o sentido): monter, descendre, sortir, rentrer, retourner, passer
   - **Être** = sem objeto direto (mouvement): *Je suis monté au 3e étage.*
   - **Avoir** = com objeto direto (action sur qqch): *J'ai monté les valises.*
3. **Tous les pronominaux** usam être: se lever → je **me suis** levé, se coucher → il **s'est** couché

### Exemplos com Concordância

| Sujeito | Exemplo | Concordância |
|---------|---------|-------------|
| je (masc.) | je suis allé | — |
| je (fem.) | je suis allé**e** | +e |
| tu (masc.) | tu es parti | — |
| tu (fem.) | tu es parti**e** | +e |
| il | il est venu | — |
| elle | elle est venu**e** | +e |
| nous (masc. ou misto) | nous sommes arrivé**s** | +s |
| nous (fem.) | nous sommes arrivé**es** | +es |
| vous (masc. sg. formel) | vous êtes resté | — |
| vous (fem. sg. formel) | vous êtes resté**e** | +e |
| vous (masc. pl.) | vous êtes resté**s** | +s |
| vous (fem. pl.) | vous êtes resté**es** | +es |
| ils | ils sont tombé**s** | +s |
| elles | elles sont tombé**es** | +es |
