## Parte 1 — Tabelas de Referência Rápida

### 1.1 Futur Simple (Lição 11)

#### Formação

```
INFINITIVO + terminações do futur
```

| Pronome | Terminação | Exemplo: PARLER |
|---------|-----------|-----------------|
| je | -ai | parler**ai** |
| tu | -as | parler**as** |
| il/elle/on | -a | parler**a** |
| nous | -ons | parler**ons** |
| vous | -ez | parler**ez** |
| ils/elles | -ont | parler**ont** |

> **Dica:** Para verbos em -RE, tire o -e final: prendr~~e~~ → je prendrai.

#### Irregulares essenciais

| Infinitivo | Radical do futur | Exemplo (je) |
|-----------|-----------------|-------------|
| être | ser- | je **serai** |
| avoir | aur- | j'**aurai** |
| faire | fer- | je **ferai** |
| aller | ir- | j'**irai** |
| pouvoir | pourr- | je **pourrai** |
| vouloir | voudr- | je **voudrai** |
| devoir | devr- | je **devrai** |
| savoir | saur- | je **saurai** |
| venir | viendr- | je **viendrai** |
| voir | verr- | je **verrai** |
| envoyer | enverr- | j'**enverrai** |
| courir | courr- | je **courrai** |

#### Futur proche vs Futur simple

| Futur proche | Futur simple | Uso |
|-------------|-------------|-----|
| Je **vais manger** | Je **mangerai** | Proche = plano concreto, imediato |
| Je **vais partir** | Je **partirai** | Simple = futuro mais distante, formal |

### 1.2 Conditionnel Présent (Lição 12)

#### Formação

```
RADICAL DO FUTUR + terminações do imparfait
```

| Pronome | Terminação | Exemplo: PARLER |
|---------|-----------|-----------------|
| je | -ais | parler**ais** |
| tu | -ais | parler**ais** |
| il/elle/on | -ait | parler**ait** |
| nous | -ions | parler**ions** |
| vous | -iez | parler**iez** |
| ils/elles | -aient | parler**aient** |

> **Dica:** Mesmo radical irregular do futur! Se **avoir** → aur- no futur, então aur- no conditionnel também: j'**aurais**.

#### Estrutura com SI

| Estrutura | Significado | Exemplo |
|-----------|------------|---------|
| Si + **présent** → **futur simple** | Condição real | Si j'ai le temps, je **jouerai** à WoW. |
| Si + **imparfait** → **conditionnel** | Condição hipotética | Si j'avais le temps, je **jouerais** à WoW. |

> **Atenção:** NUNCA use conditionnel ou futur depois de **si**! ~~Si j'aurais~~ → Si j'avais.

#### Usos do conditionnel

| Uso | Exemplo |
|-----|---------|
| Pedido educado | Je **voudrais** un café, s'il vous plaît. |
| Conselho | Tu **devrais** étudier plus. |
| Desejo | J'**aimerais** voyager en France. |
| Hipótese | Si j'étais riche, j'**achèterais** un château. |
| Informação não confirmada | Il **serait** malade. (Dizem que ele está doente.) |

### 1.3 Verbos Pronominais (Lição 13)

#### Tipos

| Tipo | Significado | Exemplo |
|------|-----------|---------|
| Reflexivo | A ação volta para o sujeito | Je **me** lave. (Eu me lavo.) |
| Recíproco | Ação mútua | Ils **se** parlent. (Eles se falam.) |
| Sentido próprio | Verbo só existe pronominal | Je **me** souviens. (Eu me lembro.) |

#### Conjugação em todos os tempos

| Tempo | Exemplo: se lever |
|-------|------------------|
| Presente | Je **me lève** |
| Passé composé | Je **me suis levé(e)** |
| Imparfait | Je **me levais** |
| Futur simple | Je **me lèverai** |
| Conditionnel | Je **me lèverais** |
| Impératif (+) | **Lève-toi !** |
| Impératif (-) | **Ne te lève pas !** |

#### Regras no passé composé

- Sempre com **être**: Je **me suis** levé.
- Concordância com o sujeito: Elle **s'est levée**. Ils **se sont levés**.
- Negação: Je **ne me suis pas** levé.

### 1.4 Verbos Modais (Lição 14)

#### Conjugação no presente

| Pronome | pouvoir | vouloir | devoir | savoir |
|---------|---------|---------|--------|--------|
| je | peux | veux | dois | sais |
| tu | peux | veux | dois | sais |
| il/elle | peut | veut | doit | sait |
| nous | pouvons | voulons | devons | savons |
| vous | pouvez | voulez | devez | savez |
| ils/elles | peuvent | veulent | doivent | savent |

#### Significados e usos

| Verbo | Significado principal | Exemplo |
|-------|----------------------|---------|
| **pouvoir** | poder (capacidade/permissão) | Je **peux** coder pendant 8 heures. |
| **vouloir** | querer | Je **veux** apprendre le français. |
| **devoir** | dever (obrigação/probabilidade) | Je **dois** finir ce projet. |
| **savoir** | saber (habilidade aprendida) | Je **sais** programmer en C#. |

#### SAVOIR vs CONNAÎTRE

| SAVOIR | CONNAÎTRE |
|--------|----------|
| saber fazer algo | conhecer (pessoa, lugar, obra) |
| + infinitivo / + que | + substantivo |
| Je **sais** nager. | Je **connais** Paris. |
| Je **sais** que c'est difficile. | Je **connais** Katya. |
| Je **sais** programmer. | Je **connais** ce jeu. |

> **Erro comum:** ~~Je sais Paris.~~ → Je **connais** Paris. / ~~Je connais nager.~~ → Je **sais** nager.

### 1.5 Pronomes Relativos (Lição 15)

#### Guia de decisão

| Pronome | Função | Substituição | Exemplo |
|---------|--------|-------------|---------|
| **qui** | sujeito | il/elle fait → **qui** fait | Le développeur **qui** travaille chez Jure est brésilien. |
| **que** | objeto direto | je vois il → **que** je vois | Le jeu **que** j'adore est Horizon. |
| **où** | lugar / tempo | là / à ce moment → **où** | La ville **où** j'habite est Juiz de Fora. |
| **dont** | complemento com DE | je parle de → **dont** je parle | Le livre **dont** je parle est le Guide du Routard Galactique. |

#### Regras importantes

- **Que** → elide antes de vogal: Le film **qu'**il a vu.
- **Qui** → NUNCA elide: Le garçon **qui** est là. (nunca ~~qu'est~~)
- **Dont** → substitui "de qui", "de quoi", "duquel": C'est le jeu **dont** j'ai besoin. (avoir besoin **de**)

### 1.6 Imperativo (Lição 18)

#### Formação

| Grupo | tu | nous | vous |
|-------|-----|------|------|
| -ER (parler) | Parle ! | Parlons ! | Parlez ! |
| -IR (finir) | Finis ! | Finissons ! | Finissez ! |
| -RE (attendre) | Attends ! | Attendons ! | Attendez ! |

> **Dica:** Para os verbos em -ER, o **tu** perde o -s! Parle! (não ~~Parles!~~). Mas o -s volta antes de **y** e **en**: Parles-**en** ! Vas-**y** !

#### Irregulares

| Verbo | tu | nous | vous |
|-------|-----|------|------|
| être | **Sois !** | **Soyons !** | **Soyez !** |
| avoir | **Aie !** | **Ayons !** | **Ayez !** |
| savoir | **Sache !** | **Sachons !** | **Sachez !** |

#### Com pronomes

| Afirmativo | Negativo |
|-----------|---------|
| **Lève-toi !** | **Ne te lève pas !** |
| **Fais-le !** | **Ne le fais pas !** |
| **Donne-moi !** | **Ne me donne pas !** |
| **Parlez-lui !** | **Ne lui parlez pas !** |

> **Regra:** Afirmativo = pronome DEPOIS do verbo (com hífen). Negativo = pronome ANTES do verbo.

### 1.7 Comparativos e Superlativos (Lição 19)

#### Comparativos

| Tipo | Estrutura | Exemplo |
|------|----------|---------|
| Superioridade | **plus** + adj/adv + **que** | WoW est **plus** complexe **que** Overwatch. |
| Inferioridade | **moins** + adj/adv + **que** | MLEM est **moins** cher **que** Horizon. |
| Igualdade | **aussi** + adj/adv + **que** | Frieren est **aussi** bon **que** Solo Leveling. |

#### Comparativos de quantidade

| Tipo | Estrutura | Exemplo |
|------|----------|---------|
| Mais | **plus de** + substantivo + **que** | J'ai **plus de** temps **que** toi. |
| Menos | **moins de** + substantivo + **que** | J'ai **moins de** jeux **que** lui. |
| Tanto | **autant de** + substantivo + **que** | J'ai **autant de** travail **que** toi. |

#### Irregulares

| Adjetivo/Advérbio | Comparativo | Superlativo |
|-------------------|------------|-------------|
| bon (bom) | **meilleur(e)** | **le/la meilleur(e)** |
| bien (bem) | **mieux** | **le mieux** |
| mauvais (ruim) | **pire** ou plus mauvais | **le/la pire** |
| mal (mal) | **pis** ou plus mal | **le pis** |

> **Erro comum:** ~~plus bon~~ → **meilleur**. ~~plus bien~~ → **mieux**.

#### Superlativos

| Tipo | Estrutura | Exemplo |
|------|----------|---------|
| Superlativo (+) | **le/la/les plus** + adj | Horizon est **le jeu le plus** beau. |
| Superlativo (-) | **le/la/les moins** + adj | C'est **le cours le moins** ennuyeux. |

---

### 1.8 Expressões de Tempo (Lição 20)

#### Guia de decisão

| Expressão | Quando usar | Tempo verbal | Exemplo |
|-----------|------------|-------------|---------|
| **depuis** | Ação que continua | **presente** | Je code **depuis** 8 ans. |
| **pendant** | Duração concluída | **passé composé** | J'ai joué **pendant** 3 heures. |
| **il y a** | Quando no passado | **passé composé** | J'ai commencé **il y a** 2 ans. |
| **dans** | Quando no futuro | **presente/futur** | Le meeting est **dans** 1 heure. |
| **en** | Tempo para completar | **qualquer** | J'ai fini **en** 10 minutes. |
| **pour** | Duração planejada | **presente/futur** | Je pars **pour** 2 semaines. |

## Parte 2 — Checklist de Erros Comuns para Lusófonos

Raphael, esta lista reúne os erros mais frequentes que brasileiros cometem nas estruturas das lições 11 a 19. Revise cada um com atenção!

### Futur Simple e Conditionnel

| Erro | Correto | Explicação |
|------|---------|------------|
| ~~Si j'**aurais** le temps...~~ | Si j'**avais** le temps... | Nunca use futur/conditionnel depois de SI |
| ~~Je **allerai**~~ | J'**irai** | Aller tem radical irregular: ir- |
| ~~Je **faisrai**~~ | Je **ferai** | Faire tem radical irregular: fer- |
| ~~Je **pouvrai**~~ | Je **pourrai** | Pouvoir: pourr- (dois R!) |

### Verbos Pronominais

| Erro | Correto | Explicação |
|------|---------|------------|
| ~~Je me suis levé**e**~~ (Raphael) | Je me suis levé | Raphael é masculino, sem -e |
| ~~Elle s'est levé~~ | Elle **s'est levée** | Com être, concordância com o sujeito |
| ~~J'**ai** me levé~~ | Je **me suis** levé | Pronominais sempre com être no PC |
| ~~Lève-te !~~ | **Lève-toi !** | Imperativo afirmativo: te → toi |

### Verbos Modais

| Erro | Correto | Explicação |
|------|---------|------------|
| ~~Je **sais** Paris.~~ | Je **connais** Paris. | Savoir = saber fazer / Connaître = conhecer |
| ~~Je **connais** nager.~~ | Je **sais** nager. | Savoir + infinitivo |
| ~~Je **connais** que c'est bon.~~ | Je **sais** que c'est bon. | Savoir + que + frase |
| ~~Je **peux** un café.~~ | Je **veux** un café. | Pouvoir = capacidade, vouloir = querer |

### Pronomes Relativos

| Erro | Correto | Explicação |
|------|---------|------------|
| ~~L'homme **que** parle~~ | L'homme **qui** parle | Sujeito = qui (ele fala) |
| ~~Le film **qui** j'aime~~ | Le film **que** j'aime | Objeto direto = que (eu amo ele) |
| ~~Le livre **que** j'ai besoin~~ | Le livre **dont** j'ai besoin | Avoir besoin **de** → dont |
| ~~La ville **que** j'habite~~ | La ville **où** j'habite | Lugar = où |

### Imperativo

| Erro | Correto | Explicação |
|------|---------|------------|
| ~~**Tu** parle !~~ | **Parle !** | Sem pronome sujeito no imperativo |
| ~~**Parles** !~~ (verbo -ER) | **Parle !** | Verbos -ER: sem -s no tu |
| ~~Ne lève-toi pas !~~ | **Ne te lève pas !** | Negação: pronome antes do verbo |
| ~~Donne-je !~~ | **Donne-moi !** | me → moi no imperativo afirmativo |

### Comparativos e Superlativos

| Erro | Correto | Explicação |
|------|---------|------------|
| ~~**plus bon**~~ | **meilleur** | Bon tem comparativo irregular |
| ~~**plus bien**~~ | **mieux** | Bien tem comparativo irregular |
| ~~aussi **de** livres que~~ | **autant de** livres que | Quantidade: autant de (não aussi de) |
| ~~le **meilleure** jeu~~ | le **meilleur** jeu | Jeu = masculino → meilleur |

### Expressões de Tempo

| Erro | Correto | Explicação |
|------|---------|------------|
| ~~J'**ai appris** depuis 6 mois~~ | J'**apprends** depuis 6 mois | Depuis + presente (ação contínua) |
| ~~Je travaille ici **pendant** 3 ans~~ | Je travaille ici **depuis** 3 ans | Ação contínua = depuis |
| ~~J'ai fini **dans** 5 min~~ | J'ai fini **en** 5 min | Tempo para completar = en |
| ~~Le meeting est **il y a** 1h~~ (futuro) | Le meeting est **dans** 1h | Futuro = dans |
