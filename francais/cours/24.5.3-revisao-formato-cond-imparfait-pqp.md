# Lição 24.5.3 — Revisão de Formato: Imparfait, Conditionnel e Plus-que-parfait

---

## Objetivos

Esta lição é **só sobre como montar** os três tempos — não sobre quando usar. Ao final, você vai:

- Saber montar o **imparfait** sem pensar (o radical e as terminações)
- Saber montar o **conditionnel** vendo a ligação com o futur e o imparfait
- **Saber montar o plus-que-parfait sem travar** — o foco principal
- Não confundir mais **avais / serai / serais / aurais** etc.

---

## A Cadeia: como esses três tempos se conectam

```
IMPARFAIT          →  fornece as TERMINAÇÕES do conditionnel
                   →  fornece a CONJUGAÇÃO do auxiliar no PQP

CONDITIONNEL       =  radical do futur + terminações do imparfait

PLUS-QUE-PARFAIT   =  auxiliar (avoir/être) NO IMPARFAIT + participe passé
```

**Tradução prática:** se você dominar **imparfait**, os outros dois caem como dominó. É por isso que vamos começar por ele.

---

## Parte 1: IMPARFAIT — o bloco fundamental

### Fórmula

```
RADICAL ( = nous no presente, sem -ons )   +   -ais, -ais, -ait, -ions, -iez, -aient
```

### Encontrar o radical (vem do "nous" do presente)

| Verbo | nous (présent) | radical | je (imparfait) |
|-------|----------------|---------|----------------|
| parler | nous parl**ons** | parl- | je parl**ais** |
| finir | nous finiss**ons** | finiss- | je finiss**ais** |
| attendre | nous attend**ons** | attend- | j'attend**ais** |
| **avoir** | nous av**ons** | **av-** | **j'av**ais** |
| **être** | (exceção!) | **ét-** | **j'ét**ais** |
| faire | nous fais**ons** | fais- | je fais**ais** |
| aller | nous all**ons** | all- | j'all**ais** |
| voir | nous voy**ons** | voy- | je voy**ais** |
| pouvoir | nous pouv**ons** | pouv- | je pouv**ais** |
| vouloir | nous voul**ons** | voul- | je voul**ais** |
| savoir | nous sav**ons** | sav- | je sav**ais** |
| devoir | nous dev**ons** | dev- | je dev**ais** |
| prendre | nous pren**ons** | pren- | je pren**ais** |
| venir | nous ven**ons** | ven- | je ven**ais** |
| boire | nous buv**ons** | buv- | je buv**ais** |

### Terminações (sempre as mesmas)

| je | tu | il/elle | nous | vous | ils/elles |
|----|-----|---------|------|------|-----------|
| **-ais** | **-ais** | **-ait** | **-ions** | **-iez** | **-aient** |

### Os dois imparfait que você MAIS precisa (porque são auxiliares do PQP)

**AVOIR imparfait** ← usado no PQP da maioria dos verbos

| je | j'**avais** |
| tu | tu **avais** |
| il/elle | il **avait** |
| nous | nous **avions** |
| vous | vous **aviez** |
| ils/elles | ils **avaient** |

**ÊTRE imparfait** ← usado no PQP dos DR MRS VANDERTRAMP e pronominais

| je | j'**étais** |
| tu | tu **étais** |
| il/elle | il **était** |
| nous | nous **étions** |
| vous | vous **étiez** |
| ils/elles | ils **étaient** |

> **🚨 GRAVE ESTES DOIS BLOCOS.** Sem eles, não tem PQP. Decorou? Toda vez que precisar de PQP, você só puxa um destes dois e cola o particípio.

## Parte 2: CONDITIONNEL — só uma "mistura" do que você já sabe

### Fórmula

```
RADICAL DO FUTUR SIMPLE   +   TERMINAÇÕES DO IMPARFAIT
```

São as MESMAS terminações do imparfait: -ais, -ais, -ait, -ions, -iez, -aient.

### Radicais (= infinitivo para verbos regulares)

Para regulares, o radical do futur **é o infinitivo inteiro** (em -re, tira o "e" final):

| Verbo | Radical | je (cond) |
|-------|---------|-----------|
| parler | parler- | je parler**ais** |
| finir | finir- | je finir**ais** |
| attendre | attendr- | j'attendr**ais** |

### Os 10 radicais irregulares (MEMORIZE — mesmos do futur)

| Verbo | Radical | je (cond) |
|-------|---------|-----------|
| être | **ser-** | je ser**ais** |
| avoir | **aur-** | j'aur**ais** |
| faire | **fer-** | je fer**ais** |
| aller | **ir-** | j'ir**ais** |
| pouvoir | **pourr-** | je pourr**ais** |
| vouloir | **voudr-** | je voudr**ais** |
| venir | **viendr-** | je viendr**ais** |
| voir | **verr-** | je verr**ais** |
| savoir | **saur-** | je saur**ais** |
| devoir | **devr-** | je devr**ais** |

## Parte 3: A confusão visual — imparfait vs conditionnel vs futur

As três formas parecem-se. **Esta tabela é a chave para não errar:**

| Tempo | Estrutura | je (parler) | je (être) |
|-------|-----------|-------------|-----------|
| Imparfait | **radical do nous** + -ais | je parl**ais** | j'**ét**ais |
| Futur simple | **infinitivo (ou rad. irreg.)** + -ai | je parler**ai** | je ser**ai** |
| Conditionnel | **infinitivo (ou rad. irreg.)** + -ais | je parler**ais** | je ser**ais** |

**Atenção visual:**
- **parl + ais** = imparfait (radical curto, sem -er)
- **parler + ai** = futur (radical = infinitivo, **-ai** sem -s)
- **parler + ais** = conditionnel (radical = infinitivo, **-ais** com -s)

**A única diferença escrita entre futur e conditionnel na 1ª pessoa é o "-s" final.** Na pronúncia, o /e/ (futur) vs /ɛ/ (cond) é quase indistinguível na fala rápida.

## Parte 4: PLUS-QUE-PARFAIT — o objetivo principal desta lição

### A fórmula em uma linha

```
PLUS-QUE-PARFAIT  =  AVOIR ou ÊTRE no IMPARFAIT  +  PARTICIPE PASSÉ
```

É **literalmente** o passé composé com o auxiliar trocado: em vez de presente, vai no imparfait.

| Tempo | Estrutura | Exemplo |
|-------|-----------|---------|
| Passé composé | aux. (présent) + PP | j'**ai** mangé / je **suis** allé |
| **Plus-que-parfait** | **aux. (imparfait) + PP** | j'**avais** mangé / j'**étais** allé |

### Passo a passo para montar QUALQUER PQP

**Passo 1:** Decida o auxiliar (avoir ou être) — mesma regra do passé composé.

**Passo 2:** Conjugue o auxiliar no **imparfait** (parte 1 desta lição).

**Passo 3:** Cole o **particípio passado** (mesmo do passé composé).

**Passo 4:** Se for **être**, faça a concordância com o sujeito.

### Exemplos com avoir (a maioria dos verbos)

| Pronome | avoir (imparfait) | PP | PQP |
|---------|-------------------|-----|------|
| je | avais | mangé | j'**avais mangé** |
| tu | avais | fini | tu **avais fini** |
| il | avait | vu | il **avait vu** |
| nous | avions | fait | nous **avions fait** |
| vous | aviez | pris | vous **aviez pris** |
| ils | avaient | dit | ils **avaient dit** |

### Exemplos com être (DR MRS VANDERTRAMP + pronominais)

| Pronome | être (imparfait) | PP (com concord.) | PQP |
|---------|------------------|-------------------|------|
| je (masc) | étais | allé | j'**étais allé** |
| je (fem) | étais | allé**e** | j'**étais allée** |
| tu (masc) | étais | parti | tu **étais parti** |
| elle | était | venu**e** | elle **était venue** |
| nous (masc) | étions | sortis | nous **étions sortis** |
| elles | étaient | montées | elles **étaient montées** |

### PQP de verbos pronominais

| Pronome | pron. | être (imp) | PP | PQP completo |
|---------|-------|------------|-----|--------------|
| je | me | étais | levé | je **m'étais levé** |
| tu | te | étais | levé | tu **t'étais levé** |
| il | s' | était | levé | il **s'était levé** |
| elle | s' | était | levée | elle **s'était levée** |
| nous (masc) | nous | étions | levés | nous **nous étions levés** |
| ils | se | étaient | levés | ils **s'étaient levés** |

## Parte 5: Tabela visual — comparando PC × PQP

A única diferença entre passé composé e plus-que-parfait é **o tempo do auxiliar**.

| Verbo | Passé composé | Plus-que-parfait |
|-------|---------------|------------------|
| manger | j'**ai** mangé | j'**avais** mangé |
| finir | tu **as** fini | tu **avais** fini |
| voir | il **a** vu | il **avait** vu |
| faire | nous **avons** fait | nous **avions** fait |
| prendre | vous **avez** pris | vous **aviez** pris |
| dire | ils **ont** dit | ils **avaient** dit |
| aller | je **suis** allé | j'**étais** allé |
| partir | elle **est** partie | elle **était** partie |
| venir | nous **sommes** venus | nous **étions** venus |
| sortir | ils **sont** sortis | ils **étaient** sortis |
| se lever | je **me suis** levé | je **m'étais** levé |
| se coucher | elle **s'est** couchée | elle **s'était** couchée |

**Truque mental:** quando for montar um PQP, **pense primeiro no passé composé** e depois só troque o auxiliar para o imparfait. *J'ai mangé* → *j'avais mangé*. *Elle est venue* → *elle était venue*.

Para o PQP funcionar, você precisa do PP de cor. Os irregulares mais usados:

| Verbo | PP | PQP (je) |
|-------|-----|----------|
| être | été | j'avais **été** |
| avoir | eu | j'avais **eu** |
| faire | fait | j'avais **fait** |
| prendre | pris | j'avais **pris** |
| voir | vu | j'avais **vu** |
| dire | dit | j'avais **dit** |
| écrire | écrit | j'avais **écrit** |
| lire | lu | j'avais **lu** |
| boire | bu | j'avais **bu** |
| pouvoir | pu | j'avais **pu** |
| vouloir | voulu | j'avais **voulu** |
| savoir | su | j'avais **su** |

## Parte 7: Os erros visuais mais comuns (e como evitá-los)

### Erro 1: confundir "j'avais" com "j'aurais"

- **j'avais** = avoir no IMPARFAIT → usado no PQP
- **j'aurais** = avoir no CONDITIONNEL → forma o conditionnel passé (depois)

Mnemônico: **av**ais (sem r) = passado / **aur**ais (com r) = hipótese.

### Erro 2: esquecer a concordância com être

| Errado | Certo |
|--------|-------|
| ~~Elle était parti.~~ | Elle était **partie**. |
| ~~Ils étaient venu.~~ | Ils étaient **venus**. |
| ~~Elles s'étaient levé.~~ | Elles s'étaient **levées**. |

### Erro 3: usar o auxiliar errado

PQP usa o **mesmo auxiliar** que o passé composé. Não invente.

- *J'**ai mangé*** → *J'**avais mangé*** ✓
- *Je **suis allé*** → *J'**étais allé*** ✓
- ~~*J'avais allé*~~ ❌ (aller usa être!)

### Erro 4: erro ortográfico em "ils avaient"

Atenção: **ils avaient**, **elles avaient**. NUNCA *"ils avaients"* (com -s no final).

### Erro 5: confundir imparfait com PC na fala

Na fala rápida:
- **j'avais** (imparfait) = duas sílabas /ʒa-vɛ/
- **j'ai eu** (passé composé) = duas sílabas /ʒe-y/

Cuidado pra não escrever um achando que ouviu o outro.

## Resumo de bolso (cola pra Anki)

```
IMPARFAIT
  = radical de "nous" no presente + -ais, -ais, -ait, -ions, -iez, -aient
  Exceção: être → ét-
  Crítico: avais / étais

CONDITIONNEL
  = radical do futur + terminações do imparfait
  Os 10 irregulares: ser-, aur-, fer-, ir-, pourr-, voudr-, viendr-, verr-, saur-, devr-

PLUS-QUE-PARFAIT
  = avoir/être no IMPARFAIT + participe passé
  Truque: pega o passé composé e troca o auxiliar (présent → imparfait)
  Ex: j'ai mangé → j'avais mangé
      je suis allé → j'étais allé
      je me suis levé → je m'étais levé
```
