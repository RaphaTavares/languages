# Lição 24.5.3 — Revisão de Formato: Imparfait, Conditionnel e Plus-que-parfait

---

## Objetivos

Esta lição é **só sobre como montar** os tempos — não sobre quando usar. Ao final, você vai:

- Saber montar o **imparfait** sem pensar (o radical e as terminações)
- Saber montar o **futur simple** (radical do infinitivo + -ai/-as/-a/-ons/-ez/-ont)
- Saber montar o **conditionnel présent** vendo a ligação com o futur e o imparfait
- Saber montar o **conditionnel passé** (aux. no cond + PP) — paralelo ao PQP
- **Saber montar o plus-que-parfait sem travar** — o foco principal
- Não confundir mais **avais / aurai / aurais / serai / serais / aurais mangé / avais mangé** etc.

---

## A Cadeia: como esses tempos se conectam

```
IMPARFAIT          →  fornece as TERMINAÇÕES do conditionnel
                   →  fornece a CONJUGAÇÃO do auxiliar no PQP

FUTUR SIMPLE       =  radical (infinitivo ou irregular) + -ai, -as, -a, -ons, -ez, -ont
                   →  fornece o RADICAL do conditionnel

CONDITIONNEL       =  radical do FUTUR + terminações do IMPARFAIT
(présent)

PLUS-QUE-PARFAIT   =  auxiliar (avoir/être) NO IMPARFAIT + participe passé

CONDITIONNEL       =  auxiliar (avoir/être) NO CONDITIONNEL + participe passé
PASSÉ                (mesma estrutura do PQP, só troca o tempo do auxiliar)
```

**Tradução prática:** o **imparfait** abastece dois tempos (cond + PQP) e o **futur** abastece um (cond). Se você dominar imparfait e futur, todo o resto cai como dominó.

## Parte 1: IMPARFAIT — o bloco fundamental

### Fórmula

```
RADICAL ( = nous no presente, sem -ons )   +   -ais, -ais, -ait, -ions, -iez, -aient
```

### Encontrar o radical (vem do "nous" do presente)

| Verbo | nous (présent) | radical | je (imparfait) |
|-------|----------------|---------|----------------|
| parler | nous parl**ons** | parl- | je parl**ais** |
| finir | nous finiss**ons** | finiss- | je finiss**ais** |
| attendre | nous attend**ons** | attend- | j'attend**ais** |
| **avoir** | nous av**ons** | **av-** | **j'av**ais** |
| **être** | (exceção!) | **ét-** | **j'ét**ais** |
| faire | nous fais**ons** | fais- | je fais**ais** |
| aller | nous all**ons** | all- | j'all**ais** |
| voir | nous voy**ons** | voy- | je voy**ais** |
| pouvoir | nous pouv**ons** | pouv- | je pouv**ais** |
| vouloir | nous voul**ons** | voul- | je voul**ais** |
| savoir | nous sav**ons** | sav- | je sav**ais** |
| devoir | nous dev**ons** | dev- | je dev**ais** |
| prendre | nous pren**ons** | pren- | je pren**ais** |
| venir | nous ven**ons** | ven- | je ven**ais** |
| boire | nous buv**ons** | buv- | je buv**ais** |

### Terminações (sempre as mesmas)

| je | tu | il/elle | nous | vous | ils/elles |
|----|-----|---------|------|------|-----------|
| **-ais** | **-ais** | **-ait** | **-ions** | **-iez** | **-aient** |

### Os dois imparfait que você MAIS precisa (porque são auxiliares do PQP)

**AVOIR imparfait** ← usado no PQP da maioria dos verbos

| je | j'**avais** |
| tu | tu **avais** |
| il/elle | il **avait** |
| nous | nous **avions** |
| vous | vous **aviez** |
| ils/elles | ils **avaient** |

**ÊTRE imparfait** ← usado no PQP dos DR MRS VANDERTRAMP e pronominais

| je | j'**étais** |
| tu | tu **étais** |
| il/elle | il **était** |
| nous | nous **étions** |
| vous | vous **étiez** |
| ils/elles | ils **étaient** |

> **🚨 GRAVE ESTES DOIS BLOCOS.** Sem eles, não tem PQP. Decorou? Toda vez que precisar de PQP, você só puxa um destes dois e cola o particípio.

## Parte 2: FUTUR SIMPLE — o radical que o conditionnel também usa

### Fórmula

```
RADICAL DO FUTUR ( = infinitivo, ou radical irregular )   +   -ai, -as, -a, -ons, -ez, -ont
```

### Terminações (memorize — são quase o verbo "avoir" no presente!)

| je | tu | il/elle | nous | vous | ils/elles |
|----|-----|---------|------|------|-----------|
| **-ai** | **-as** | **-a** | **-ons** | **-ez** | **-ont** |

> **Truque mnemônico:** as terminações são **j'ai, tu as, il a, (av)ons, (av)ez, (av)ont** — o auxiliar **avoir** no presente, sem o "av-" das três últimas. Por isso muita gente fala que "o futur é avoir grudado no infinitivo".

### O radical para verbos regulares = infinitivo inteiro

Para regulares, é só colar as terminações no infinitivo (verbos em -re tiram o "e" final):

| Verbo | Radical | je (futur) |
|-------|---------|------------|
| parler | parler- | je parler**ai** |
| finir | finir- | je finir**ai** |
| attendre | attendr- | j'attendr**ai** |

### Os 10 radicais irregulares (decorar — valem TAMBÉM para o conditionnel)

| Verbo | Radical | je (futur) | je (cond) |
|-------|---------|------------|-----------|
| être | **ser-** | je ser**ai** | je ser**ais** |
| avoir | **aur-** | j'aur**ai** | j'aur**ais** |
| faire | **fer-** | je fer**ai** | je fer**ais** |
| aller | **ir-** | j'ir**ai** | j'ir**ais** |
| pouvoir | **pourr-** | je pourr**ai** | je pourr**ais** |
| vouloir | **voudr-** | je voudr**ai** | je voudr**ais** |
| venir | **viendr-** | je viendr**ai** | je viendr**ais** |
| voir | **verr-** | je verr**ai** | je verr**ais** |
| savoir | **saur-** | je saur**ai** | je saur**ais** |
| devoir | **devr-** | je devr**ai** | je devr**ais** |

### Conjugação completa: PARLER no futur

| Pronome | Futur |
|---------|-------|
| je | parler**ai** |
| tu | parler**as** |
| il/elle | parler**a** |
| nous | parler**ons** |
| vous | parler**ez** |
| ils/elles | parler**ont** |

### Conjugação completa: ÊTRE no futur (radical irregular ser-)

| Pronome | Futur |
|---------|-------|
| je | ser**ai** |
| tu | ser**as** |
| il/elle | ser**a** |
| nous | ser**ons** |
| vous | ser**ez** |
| ils/elles | ser**ont** |

## Parte 3: CONDITIONNEL PRÉSENT — só uma "mistura" do que você já sabe

### Fórmula

```
RADICAL DO FUTUR SIMPLE   +   TERMINAÇÕES DO IMPARFAIT
```

### Terminações (idênticas ao imparfait)

| je | tu | il/elle | nous | vous | ils/elles |
|----|-----|---------|------|------|-----------|
| **-ais** | **-ais** | **-ait** | **-ions** | **-iez** | **-aient** |

### O mesmo radical do futur — só muda o final

Se você acabou de aprender o futur (parte 2), o conditionnel é trivial: pega o mesmo radical e troca **-ai, -as, -a, -ons, -ez, -ont** por **-ais, -ais, -ait, -ions, -iez, -aient**.

| Verbo | Radical | je (futur) | je (cond) |
|-------|---------|------------|-----------|
| parler | parler- | je parler**ai** | je parler**ais** |
| finir | finir- | je finir**ai** | je finir**ais** |
| attendre | attendr- | j'attendr**ai** | j'attendr**ais** |
| être | ser- | je ser**ai** | je ser**ais** |
| avoir | aur- | j'aur**ai** | j'aur**ais** |
| faire | fer- | je fer**ai** | je fer**ais** |

### Conjugação completa: PARLER no conditionnel présent

| Pronome | Conditionnel |
|---------|--------------|
| je | parler**ais** |
| tu | parler**ais** |
| il/elle | parler**ait** |
| nous | parler**ions** |
| vous | parler**iez** |
| ils/elles | parler**aient** |

### Conjugação completa de AVOIR e ÊTRE no conditionnel (você vai precisar para o conditionnel passé!)

**AVOIR** (radical irregular **aur-**)

| Pronome | Conditionnel |
|---------|--------------|
| j' | aur**ais** |
| tu | aur**ais** |
| il/elle | aur**ait** |
| nous | aur**ions** |
| vous | aur**iez** |
| ils/elles | aur**aient** |

**ÊTRE** (radical irregular **ser-**)

| Pronome | Conditionnel |
|---------|--------------|
| je | ser**ais** |
| tu | ser**ais** |
| il/elle | ser**ait** |
| nous | ser**ions** |
| vous | ser**iez** |
| ils/elles | ser**aient** |

> **🚨 GRAVE ESTES DOIS BLOCOS.** Eles formam o conditionnel **passé** (próxima seção) — exatamente como `avais/étais` formam o PQP.

> **Resumo visual:** os 10 radicais irregulares (ser-, aur-, fer-, ir-, pourr-, voudr-, viendr-, verr-, saur-, devr-) servem para **futur** E **conditionnel** ao mesmo tempo. Decorou uma vez, vale para os dois tempos.

## Parte 3b: CONDITIONNEL PASSÉ — "teria feito / teria sido"

### A fórmula em uma linha

```
CONDITIONNEL PASSÉ  =  AVOIR ou ÊTRE no CONDITIONNEL  +  PARTICIPE PASSÉ
```

É **exatamente a mesma estrutura do PQP**, só que o auxiliar vai no conditionnel em vez de imparfait. Compare:

| Tempo | Estrutura | Exemplo |
|-------|-----------|---------|
| Passé composé | aux. (présent) + PP | j'**ai** mangé |
| Plus-que-parfait | aux. (imparfait) + PP | j'**avais** mangé |
| **Conditionnel passé** | **aux. (conditionnel) + PP** | j'**aurais** mangé |

### Passo a passo para montar QUALQUER conditionnel passé

**Passo 1:** Decida o auxiliar (avoir ou être) — mesma regra do passé composé.

**Passo 2:** Conjugue o auxiliar no **conditionnel** (tabelas acima).

**Passo 3:** Cole o **particípio passado**.

**Passo 4:** Se for **être**, faça a concordância com o sujeito.

### Exemplos com avoir (a maioria dos verbos)

| Pronome | avoir (cond) | PP | Conditionnel passé |
|---------|--------------|-----|---------------------|
| j' | aurais | mangé | j'**aurais mangé** |
| tu | aurais | fini | tu **aurais fini** |
| il | aurait | vu | il **aurait vu** |
| nous | aurions | fait | nous **aurions fait** |
| vous | auriez | pris | vous **auriez pris** |
| ils | auraient | dit | ils **auraient dit** |

### Exemplos com être (DR MRS VANDERTRAMP + pronominais)

| Pronome | être (cond) | PP (com concord.) | Conditionnel passé |
|---------|-------------|-------------------|---------------------|
| je (masc) | serais | allé | je **serais allé** |
| je (fem) | serais | allé**e** | je **serais allée** |
| tu (masc) | serais | parti | tu **serais parti** |
| elle | serait | venu**e** | elle **serait venue** |
| nous (masc) | serions | sortis | nous **serions sortis** |
| elles | seraient | montées | elles **seraient montées** |

### Pronominais no conditionnel passé

| Pronome | pron. | être (cond) | PP | Conditionnel passé |
|---------|-------|-------------|-----|---------------------|
| je | me | serais | levé | je **me serais levé** |
| tu | te | serais | levé | tu **te serais levé** |
| il | s' | serait | levé | il **se serait levé** |
| elle | s' | serait | levée | elle **se serait levée** |
| nous (masc) | nous | serions | levés | nous **nous serions levés** |
| ils | se | seraient | levés | ils **se seraient levés** |

### Tabela visual — PC × PQP × Conditionnel passé

| Verbo | Passé composé | Plus-que-parfait | Conditionnel passé |
|-------|---------------|------------------|---------------------|
| manger | j'**ai** mangé | j'**avais** mangé | j'**aurais** mangé |
| voir | il **a** vu | il **avait** vu | il **aurait** vu |
| faire | nous **avons** fait | nous **avions** fait | nous **aurions** fait |
| aller | je **suis** allé | j'**étais** allé | je **serais** allé |
| venir | elle **est** venue | elle **était** venue | elle **serait** venue |
| se lever | je **me suis** levé | je **m'étais** levé | je **me serais** levé |

**Truque mental:** os três tempos compostos (PC, PQP, cond passé) só se diferenciam pelo **tempo do auxiliar**. O participe passé não muda. Mude só o "ai/avais/aurais" e pronto.

> **Tradução prática:**
> - PC: "eu comi"
> - PQP: "eu tinha comido"
> - Conditionnel passé: "eu teria comido"

### Uso curto (revisão de sentido)

O conditionnel passé aparece principalmente nas **hipóteses irreais no passado** com `si + plus-que-parfait`:

- *Si j'**avais eu** le temps, j'**aurais joué** au jiu-jitsu hier.* (Se eu tivesse tido tempo, teria treinado jiu-jitsu ontem.)
- *Si tu **étais venu**, on **serait sortis** ensemble.* (Se você tivesse vindo, teríamos saído juntos.)

A fórmula completa do "L3" do si condicional: **si + PQP → conditionnel passé**.

## Parte 4: A confusão visual — imparfait vs futur vs conditionnel

As três formas parecem-se. **Esta tabela é a chave para não errar:**

| Tempo | Estrutura | je (parler) | je (être) |
|-------|-----------|-------------|-----------|
| Imparfait | **radical do nous** + -ais | je parl**ais** | j'**ét**ais |
| Futur simple | **infinitivo (ou rad. irreg.)** + -ai | je parler**ai** | je ser**ai** |
| Conditionnel | **infinitivo (ou rad. irreg.)** + -ais | je parler**ais** | je ser**ais** |

**Atenção visual:**
- **parl + ais** = imparfait (radical curto, sem -er)
- **parler + ai** = futur (radical = infinitivo, **-ai** sem -s)
- **parler + ais** = conditionnel (radical = infinitivo, **-ais** com -s)

**A única diferença escrita entre futur e conditionnel na 1ª pessoa é o "-s" final.** Na pronúncia, o /e/ (futur) vs /ɛ/ (cond) é quase indistinguível na fala rápida.

## Parte 5: PLUS-QUE-PARFAIT — o objetivo principal desta lição

### A fórmula em uma linha

```
PLUS-QUE-PARFAIT  =  AVOIR ou ÊTRE no IMPARFAIT  +  PARTICIPE PASSÉ
```

É **literalmente** o passé composé com o auxiliar trocado: em vez de presente, vai no imparfait.

| Tempo | Estrutura | Exemplo |
|-------|-----------|---------|
| Passé composé | aux. (présent) + PP | j'**ai** mangé / je **suis** allé |
| **Plus-que-parfait** | **aux. (imparfait) + PP** | j'**avais** mangé / j'**étais** allé |

### Passo a passo para montar QUALQUER PQP

**Passo 1:** Decida o auxiliar (avoir ou être) — mesma regra do passé composé.

**Passo 2:** Conjugue o auxiliar no **imparfait** (parte 1 desta lição).

**Passo 3:** Cole o **particípio passado** (mesmo do passé composé).

**Passo 4:** Se for **être**, faça a concordância com o sujeito.

### Exemplos com avoir (a maioria dos verbos)

| Pronome | avoir (imparfait) | PP | PQP |
|---------|-------------------|-----|------|
| je | avais | mangé | j'**avais mangé** |
| tu | avais | fini | tu **avais fini** |
| il | avait | vu | il **avait vu** |
| nous | avions | fait | nous **avions fait** |
| vous | aviez | pris | vous **aviez pris** |
| ils | avaient | dit | ils **avaient dit** |

### Exemplos com être (DR MRS VANDERTRAMP + pronominais)

| Pronome | être (imparfait) | PP (com concord.) | PQP |
|---------|------------------|-------------------|------|
| je (masc) | étais | allé | j'**étais allé** |
| je (fem) | étais | allé**e** | j'**étais allée** |
| tu (masc) | étais | parti | tu **étais parti** |
| elle | était | venu**e** | elle **était venue** |
| nous (masc) | étions | sortis | nous **étions sortis** |
| elles | étaient | montées | elles **étaient montées** |

### PQP de verbos pronominais

| Pronome | pron. | être (imp) | PP | PQP completo |
|---------|-------|------------|-----|--------------|
| je | me | étais | levé | je **m'étais levé** |
| tu | te | étais | levé | tu **t'étais levé** |
| il | s' | était | levé | il **s'était levé** |
| elle | s' | était | levée | elle **s'était levée** |
| nous (masc) | nous | étions | levés | nous **nous étions levés** |
| ils | se | étaient | levés | ils **s'étaient levés** |

## Parte 6: Tabela visual — comparando PC × PQP

A única diferença entre passé composé e plus-que-parfait é **o tempo do auxiliar**.

| Verbo | Passé composé | Plus-que-parfait |
|-------|---------------|------------------|
| manger | j'**ai** mangé | j'**avais** mangé |
| finir | tu **as** fini | tu **avais** fini |
| voir | il **a** vu | il **avait** vu |
| faire | nous **avons** fait | nous **avions** fait |
| prendre | vous **avez** pris | vous **aviez** pris |
| dire | ils **ont** dit | ils **avaient** dit |
| aller | je **suis** allé | j'**étais** allé |
| partir | elle **est** partie | elle **était** partie |
| venir | nous **sommes** venus | nous **étions** venus |
| sortir | ils **sont** sortis | ils **étaient** sortis |
| se lever | je **me suis** levé | je **m'étais** levé |
| se coucher | elle **s'est** couchée | elle **s'était** couchée |

**Truque mental:** quando for montar um PQP, **pense primeiro no passé composé** e depois só troque o auxiliar para o imparfait. *J'ai mangé* → *j'avais mangé*. *Elle est venue* → *elle était venue*.

## Parte 8: Os erros visuais mais comuns (e como evitá-los)

### Erro 1: confundir "j'avais" com "j'aurais"

- **j'avais** = avoir no IMPARFAIT → usado no PQP
- **j'aurais** = avoir no CONDITIONNEL → forma o conditionnel passé (depois)

Mnemônico: **av**ais (sem r) = passado / **aur**ais (com r) = hipótese.

### Erro 2: esquecer a concordância com être

| Errado | Certo |
|--------|-------|
| ~~Elle était parti.~~ | Elle était **partie**. |
| ~~Ils étaient venu.~~ | Ils étaient **venus**. |
| ~~Elles s'étaient levé.~~ | Elles s'étaient **levées**. |

### Erro 3: usar o auxiliar errado

PQP usa o **mesmo auxiliar** que o passé composé. Não invente.

- *J'**ai mangé*** → *J'**avais mangé*** ✓
- *Je **suis allé*** → *J'**étais allé*** ✓
- ~~*J'avais allé*~~ ❌ (aller usa être!)

### Erro 4: erro ortográfico em "ils avaient"

Atenção: **ils avaient**, **elles avaient**. NUNCA *"ils avaients"* (com -s no final).

### Erro 5: confundir imparfait com PC na fala

Na fala rápida:
- **j'avais** (imparfait) = duas sílabas /ʒa-vɛ/
- **j'ai eu** (passé composé) = duas sílabas /ʒe-y/

Cuidado pra não escrever um achando que ouviu o outro.

## Resumo de bolso (cola pra Anki)

```
IMPARFAIT
  = radical de "nous" no presente + -ais, -ais, -ait, -ions, -iez, -aient
  Exceção: être → ét-
  Crítico: avais / étais

FUTUR SIMPLE
  = radical (infinitivo, ou irregular) + -ai, -as, -a, -ons, -ez, -ont
  Truque: as terminações são quase o "avoir" no presente (ai, as, a, ons, ez, ont)
  Os 10 irregulares: ser-, aur-, fer-, ir-, pourr-, voudr-, viendr-, verr-, saur-, devr-

CONDITIONNEL PRÉSENT
  = MESMO RADICAL do futur + TERMINAÇÕES DO IMPARFAIT
  Diferença para futur: troca -ai/-as/-a/-ons/-ez/-ont por -ais/-ais/-ait/-ions/-iez/-aient
  Ex: futur "je serai" → cond "je serais" (só ganha um -s)
  Crítico: aurais / serais (usados no conditionnel passé)

PLUS-QUE-PARFAIT
  = avoir/être no IMPARFAIT + participe passé
  Truque: pega o passé composé e troca o auxiliar (présent → imparfait)
  Ex: j'ai mangé → j'avais mangé
      je suis allé → j'étais allé
      je me suis levé → je m'étais levé

CONDITIONNEL PASSÉ
  = avoir/être no CONDITIONNEL + participe passé
  Truque: mesma fórmula do PQP, só troca o tempo do auxiliar
  Ex: j'avais mangé (PQP) → j'aurais mangé (cond passé) = "teria comido"
      j'étais allé (PQP) → je serais allé (cond passé) = "teria ido"
      je m'étais levé (PQP) → je me serais levé (cond passé) = "teria me levantado"

OS TRÊS TEMPOS COMPOSTOS LADO A LADO
  PC:           j'ai mangé        / je suis allé        / je me suis levé
  PQP:          j'avais mangé     / j'étais allé        / je m'étais levé
  Cond passé:   j'aurais mangé    / je serais allé      / je me serais levé
  → Mude SÓ o tempo do auxiliar. O particípio é o mesmo.
```
